3. Chrome Extension — Data Practices

What the extension does

The SmartStack Chrome Extension automatically attaches an Amazon affiliate tag to Amazon website visits. When you visit Amazon.com or other Amazon domains, the extension appends ?tag=smartstacksup-20 to the URL. This means any purchases you make on Amazon support SmartStack Supplements at no additional cost to you.

Local storage only

The extension stores a single counter (number of Amazon visits tagged) locally in your browser using Chrome's built-in storage API. This data:

  • Never leaves your device
  • Is never transmitted to our servers or any third party
  • Can be cleared at any time by removing the extension
  • Is only used to display your visit count in the extension popup

Permissions used

  • tabs — Used to detect Amazon page navigation so the affiliate tag can be applied
  • activeTab — Used to read the current tab URL to check if it is an Amazon domain
  • storage — Used to save your local visit counter only
  • host permissions (Amazon domains) — Required to modify Amazon URLs to include the affiliate tag

No remote code execution

The extension does not load or execute any remote code. All functionality is contained within the extension package itself.
